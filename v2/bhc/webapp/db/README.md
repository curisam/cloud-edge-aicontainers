# 데이터 생성, 조작 및 DB 구축
개별 엣지 노드 모니터링, 구축된 모델 배포 및 추론 성능 분석, 시스템 이상징후 탐지 등의 목적으로 다양한 데이터를 생성, 수집, 보관해야 합니다.<br>
다양한 방법, 기술을 통해 데이터를 생성하고 보관하는 실험을 수행합니다.<br>
구축된 데이터를 grafana와 연결하여 시각화하는 방안을 함께 모색합니다.

## 개별 엣지 노드 정보 추출 및 시각화
